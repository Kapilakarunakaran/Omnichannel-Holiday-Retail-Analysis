# 📖 Project Overview

This project was developed for the **FP20 Analytics × ZoomCharts Mini Challenge**, where the objective was to transform raw omnichannel retail transaction data into an interactive business intelligence solution.

The dashboard provides a comprehensive view of sales performance, product profitability, promotional effectiveness, delivery operations, and customer return behavior. By combining multiple business perspectives into a single report, it enables stakeholders to identify revenue opportunities, operational bottlenecks, and customer purchasing trends through interactive analytics.

---

# 🎯 Business Objectives

This dashboard was designed to help business leaders answer strategic questions such as:

- Which product categories generate the highest revenue and profitability?
- Which promotional campaigns deliver the greatest return on investment?
- Which customer segments contribute the most promotional revenue?
- Which delivery partners perform most efficiently?
- Which products have the highest return rates?
- What are the leading reasons customers return products?

# 💡 Key Business Insights

- Grocery products generated the highest promotional revenue during the holiday season.
- Holiday campaigns delivered the strongest promotional ROI.
- Returning customers contributed a significant share of promotional revenue.
- Wrong Size was the leading reason for customer returns.
- Most deliveries were completed within the expected delivery window.
- Critical delivery delays represented only a small percentage of shipments.

---

# 💼 Business Value

This dashboard enables retail decision-makers to:

- Improve promotional strategy using campaign ROI analysis.
- Reduce refund costs by identifying major return drivers.
- Monitor carrier performance to improve delivery efficiency.
- Identify profitable products and categories for strategic investment.
- Support executive decision-making through interactive KPI reporting.
